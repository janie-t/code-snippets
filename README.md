#Reverse a string
```
function FirstReverse(str) { 
    var splitString = str.split("");
    var reverseArray = splitString.reverse();
    var joinArray = reverseArray.join("");
  return joinArray;     
}
// keep this function call here 
FirstReverse('Hi my name is Janie');```
